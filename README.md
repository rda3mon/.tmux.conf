# tmux
tmux configuration

### Installation

1) Clone the repository

`git clone git@github.com:rda3mon/tmux.git ~/.tmux`

2) Install tmux

`ln -s ~/.tmux/tmux.conf ~/.tmux.conf && tmux source-file ~/.tmux.conf`
